title: Как это работает?
description: Автоматическая лента истории обновлений для пользователей
icon: code
---

Вам не нужно обновлять Changelog вручную. Процесс полностью автоматизирован:

<Steps>
  <Step title="Создание версии">
    В момент выпуска релиза (раздел **Версии**) вы привязываете к нему **Пост** с описанием изменений.
  </Step>
  <Step title="Публикация">
    Releazio автоматически добавляет этот пост в общую ленту новостей.
  </Step>
  <Step title="Просмотр">
    Пользователи мгновенно видят обновление через виджет внутри приложения или по прямой веб-ссылке.
  </Step>
</Steps>
