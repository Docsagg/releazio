Чтобы начать собирать фидбек, нужно настроить логику виджета. Нажмите кнопку **«Создать опрос»**.

<Warning>
  **Требование к активации**
  
  Чтобы виджет заработал, необходимо:
  1. Завершить [Интеграцию](/integrations) и добавить API Key в проект.
  2. Заполнить все обязательные поля (отмечены `*`).
  3. Включить переключатель **«Активировать виджет»**.
</Warning>

### 1. Основные настройки

Тексты, которые увидит пользователь.

<ResponseField name="Заголовок окна" type="Текст (Обязательно)">
  Главный вопрос (например: *"Вам нравится новое обновление?"*).
</ResponseField>

<ResponseField name="Подзаголовок (Оценка ≤ 3)" type="Текст">
  Сообщение для недовольных пользователей. Используйте это, чтобы узнать причину негатива, не отправляя их в стор (например: *"Что нам улучшить?"*).
</ResponseField>

<ResponseField name="Подзаголовок (Оценка ≥ 4)" type="Текст">
  Сообщение для лояльных пользователей (например: *"Спасибо! Мы старались"*).
</ResponseField>

<ResponseField name="Сообщение после отправки" type="Текст (Обязательно)">
  Благодарность после того, как пользователь нажал кнопку отправить.
</ResponseField>

### 2. Внешний вид и логика

Настройка поведения самого виджета.

<ResponseField name="Иконка шкалы" type="Выбор">
  Выберите стиль звезд или смайликов (3 варианта).
</ResponseField>

<ResponseField name="Поле комментария" type="Чекбокс">
  Если включено, пользователь сможет написать текстовый отзыв вместе с оценкой.
</ResponseField>

<ResponseField name="Оценить на внешней площадке" type="Чекбокс">
  **Важная функция:** Перенаправляет довольных пользователей в App Store или Google Play, повышая ваш публичный рейтинг.
</ResponseField>

### 3. Таймауты показа

Чтобы не надоедать пользователям, настройте частоту появления окна.

<ResponseField name="Таймаут без отправки" type="Минуты">
  Через сколько показать виджет снова, если пользователь просто закрыл его.
</ResponseField>

<ResponseField name="Таймаут после отправки" type="Минуты">
  Через сколько запросить отзыв повторно, если пользователь уже проголосовал (например, в следующей версии).
</ResponseField>
